Travel Website - Universal Orlando Resort

This project aims to create a modern, fully-responsive website that delivers an optimal viewing experience across all devices from desktop computers to mobile phones. The site utilizes HTML5 for structured content organization, CSS3 for styling with flexbox and grid layouts that automatically adjust based on screen size, and media queries to target specific viewport dimensions. The development process follows a mobile-first approach, beginning with the smallest screen size and progressively enhancing the experience for larger displays.
